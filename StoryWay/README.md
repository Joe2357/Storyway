# StoryWay

> 2020 - 2학기 Algorithm Term Project  
> By using Java

## 용도

- 입력된 상점들을 돌아볼 수 있는 가장 좋은 경로를 출력

## 사용된 알고리즘

- Floyd-Washall
  - group으로 묶인 정점들을 이동하기 위한 인접행렬을 만드는 용도
- Backtracking
  - 가능한 모든 경로를 검색하는 데 사용
  - 더 이상 불가능한 경로라면 이후의 값을 탐색하지 않음
